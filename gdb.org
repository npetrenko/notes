* Startup
** loading scripts
   Parameter `-...x` controls execution of specific script: before or after `~/.gdbinit`, before or after loading the program
   `set auto-load local-gdbinit` -- permits to load local scripts
   
* Logging
  Most useful:
  - `set logging on`
  - `set logging file ...`
  - `show logging`
